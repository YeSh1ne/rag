# 论文 RAG 问答系统

---

## 项目简介

本项目以 40 篇大模型与引文评估相关学术论文为数据源，搭建轻量化本地 RAG 问答框架。可接收自然语言中文问题，从论文知识库中匹配相关文本片段，结合大模型生成作答内容，同时输出**论文名称、页码、chunk\_id**三级引用来源，满足文献查阅、论文研读、问题溯源等使用场景。

## 技术选型

|  序号 | 流程环节      | 环节说明                      | 首选方案                | 备选方案                     |
| :-: | :-------- | :------------------------ | :------------------ | :----------------------- |
|  1  | 文档解析      | 将原始文档（PDF/Word 等）转换为结构化文本 | Marker              | LlamaParse (API), MinerU |
|  2  | 文本分片      | 按语义结构或固定长度切分文本块           | RAG 框架内置分割器         | 自定义分片策略                  |
|  3  | Embedding | 将文本块转换为向量表示               | BGE-M3              | **Qwen3-Embedding**      |
|  4  | 向量存储      | 存储文本向量及元数据                | Milvus / Chroma     | Weaviate, Qdrant         |
|  5  | 检索召回      | 根据查询向量召回相似文本块             | Embedding + 向量库     | -                        |
|  6  | 重排序       | 对召回结果进行精排优化               | bge-reranker-v2-m3  | Cohere Rerank v3         |
|  7  | 答案生成      | 大模型基于检索结果生成回答             | Qwen2.5-72B / GLM-4 | DeepSeek-V3, GPT-4o      |
|  8  | 评估优化      | 评估系统效果并持续优化               | RAGAS               | TruLens, 人工评估            |
|  -  | 框架支撑      | 串联全流程的工程骨架                | LlamaIndex          | LangChain, RAGFlow       |
